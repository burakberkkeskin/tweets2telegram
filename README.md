# tweets2telegram

Listenes For The Spesific "Crypto News" list and sends new tweets as message to subscribers.

t.me/cryptonewsalerts_bot

Change the list id as your wish. 

Create a secret.json file and fill the details: 

{
    "telegram_api_key":" ",
    "consumer_key":" ",
    "consumer_secret":" ",
    "access_token":" ",
    "access_token_secret":" ",
}
